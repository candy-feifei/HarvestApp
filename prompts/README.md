# 团队通用 Prompt 模板（HarvestApp / AI 技能大赛）

本目录存放**脱敏、可复用**的 Prompt 骨架，供需求、设计、开发、测试、文档等环节统一口径；与交付物中的《AI 使用说明》配合，便于评委查看 Git 记录与迭代过程。

## 使用约定

1. **复制模板**：不要直接在生产对话里贴含真实客户/内网/密钥的原文；先复制本目录对应文件，再填入「上下文」占位符。
2. **脱敏后进库**：可提交「模板 + 已脱敏示例」；完整对话导出若含敏感信息，仅放内网或附件，仓库中保留节选与说明链接即可。
3. **与 PR 对应**：重大功能建议在 PR 描述中注明「使用的模板文件名 + 人工校验点」，方便复盘。
4. **迭代模板**：若某类任务发现更好的写法，直接改对应 `.md` 并走 PR，形成团队资产。
